# booknote
***
## 项目介绍

个人平时喜好读书, 但读过的书有些重要的事情需要记录, 以备后用.
读书时最忌讳十分钟热血, 读过便忘记, 因此需要记录读书当时的心得和知识点等等.
这样自己的印象会比单纯读书深刻很多.
> 注: 最好使用vscode+markdownPreview+markdownMath插件查看

## 目录介绍

### datastructure
数据结构方面的书籍笔记

### english
英语学习方面的书籍笔记, 单词, 应用的场景等等

### math
数学学习的笔记, 感悟等等

### networks
计算机网络方面的笔记

### OS
操作系统方面的笔记

### programming language
编程语言及语言的框架, 类库等方面的笔记

### script
各种脚本语言的教程和练习
